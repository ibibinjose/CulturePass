CulturePass is a lifestyle app for cultural diaspora communities worldwide.

Key flows to test:
1. Sign up with email → select location → join a community
2. Browse events on Discover tab → tap an event → view event detail
3. Tap the location pill in the top bar → select a different city
4. Navigate to Calendar tab → select a date with events
5. Navigate to Perks tab → browse available perks

Stripe payments use test mode during review. Use card 4242 4242 4242 4242
to complete a test ticket purchase.

Location permission: The app requests location access to detect the user's
